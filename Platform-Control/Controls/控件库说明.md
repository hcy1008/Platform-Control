## 🎨 控件特性总览

### 1️⃣ **LoadPortControl** - 料盒端口控件
- ✅ 25片Wafer状态网格显示（5×5布局）
- ✅ 料盒ID、到位/夹紧状态
- ✅ OK/NG/加工中统计
- ✅ 默认数据：23个OK + 1个NG + 1个Processing

### 2️⃣ **ChamberControl** - 腔体控件
- ✅ 真空度显示（科学计数法）
- ✅ 温度、射频功率、气体流量、压力
- ✅ 运行时间
- ✅ 报警/运行LED指示
- ✅ 默认数据：完整运行参数

### 3️⃣ **PipeControl** - 管路控件（已升级）
- ✅ 压力、流量、温度显示
- ✅ 集成阀门控件，可控制开关
- ✅ 流水动画效果（开启时显示流动效果）
- ✅ 阀门状态实时反馈
- ✅ 默认数据：正常流动状态

### 4️⃣ **ValveControl** - 阀门控件（新增）
- ✅ 阀门开关状态显示
- ✅ 开启状态：绿色，手柄向下
- ✅ 关闭状态：红色，手柄向上
- ✅ 阀门名称显示
- ✅ 状态文字提示

### 5️⃣ **RobotControl** - 机械臂控件
- ✅ X/Y/Z轴位置显示
- ✅ 旋转角度、速度
- ✅ 有片检测
- ✅ 电源/伺服/报警LED
- ✅ 默认数据：运行中状态

### 6️⃣ **StatusIndicator** - 状态指示灯
- ✅ 多状态支持（OK/警告/报警/离线/信息）
- ✅ 发光效果
- ✅ 通用场景

### 7️⃣ **ValueDisplay** - 数值显示
- ✅ 参数名称/数值/单位
- ✅ 可配置小数位数
- ✅ 卡片式布局

## 📦 如何扩展新控件

### 步骤1：创建控件文件
在对应分类目录下创建：
```
Controls/[分类]/YourControl.xaml
Controls/[分类]/YourControl.xaml.cs
```

### 步骤2：定义依赖属性
```csharp
public string YourProperty
{
    get => (string)GetValue(YourPropertyProperty);
    set => SetValue(YourPropertyProperty, value);
}
public static readonly DependencyProperty YourPropertyProperty =
    DependencyProperty.Register("YourProperty", typeof(string), typeof(YourControl), 
        new PropertyMetadata("默认值", OnYourPropertyChanged));
```

### 步骤3：在XAML中绑定
```xml
<TextBlock Text="{Binding YourProperty, RelativeSource={RelativeSource AncestorType=UserControl}}"/>
```

### 步骤4：在MainWindow中使用
```xml
<Window xmlns:your="clr-namespace:Platform_Control.Controls.YourNamespace">
    <your:YourControl YourProperty="Value"/>
</Window>
```

## 📖 管路监控系统使用说明

### 功能特点
- ✅ 主页点击"管路监控"按钮即可进入管路监控页面
- ✅ 每条管路都集成阀门控件，可独立控制开关
- ✅ 阀门开启时显示蓝色流水动画效果
- ✅ 阀门关闭时流水效果消失，显示红色状态
- ✅ 实时显示压力、流量、温度参数
- ✅ 控制面板支持单条管路控制
- ✅ 支持全部开启/全部关闭批量操作

### 使用方法
1. 启动程序后，在主页点击"管路监控"按钮进入管路页面
2. 在管路列表中可以看到5条管路及其状态
3. 点击控制面板中的"开启"/"关闭"按钮控制对应管路
4. 或使用底部"全部开启"/"全部关闭"按钮进行批量操作
5. 点击"返回主页"按钮返回主页

### 控件属性绑定
```xml
<pipeline:PipeControl 
    PipeName="管路名称"
    ValveName="阀门名称"
    IsValveOpen="True/False"
    Pressure="0.5 MPa"
    FlowRate="10.5 L/min"
    Temperature="25.0 °C"/>
```
